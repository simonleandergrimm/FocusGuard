# Security policy

FocusGuard installs a root-owned helper and modifies a marked section of `/etc/hosts`, so security reports are taken seriously.

Please do not open a public issue for a vulnerability. Use GitHub's **Report a vulnerability** flow in the repository's Security tab to send a private report. Include the affected version or commit, macOS version, reproduction steps, and impact. Do not include a real OpenAI API key or other personal data.

The current source-build beta does not publish signed release binaries. Only source from this repository is supported.
